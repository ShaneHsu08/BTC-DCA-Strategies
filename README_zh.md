# 比特币定投策略比较工具

[English](README.md)

<div align="center"><h1><a href=https://dca.btc.sv>在线演示</a></h1></div>

一个基于React的Web应用程序，用于比较不同的比特币定投策略。支持标准定投、动态定投和价值平均法三种策略的模拟和比较。

## 关于定投

### 为何定投？
- 看好BTC的长期价值，但希望避免一次性投入带来的择时风险，分散买入时点，降低整体风险。
- 看好BTC的长期价值，但由于资金有限、希望逐步建仓。
- 虽然看不懂BTC，但希望利用BTC的波动性以较低的价格持有一些BTC。
...

### 三种策略
1. 标准定投策略：固定金额定期投资，比如每周固定投入500美元。
- 优点：最为简单，多数交易所直接支持，只要固定时间打钱就行了。
- 缺点：不管现在价格是高是低都投一样的钱，成本会高一些。
2. 动态定投策略：在标准定投基础上加上一点择时操作，最常见的是使用RSI指标，大多数人都在买入的时候我少买一点，大多数人都在卖出的时候我多买一点。
- 优点：平均买入价格会低一些，市场下跌时回撤会小一些。
- 缺点：每次定投需要计算一下投入多少，需要额外的程序。
3. 价值平均策略（VA）：定期增加固定的持币价值，比如每周持币价值固定增加500美元，币价跌了，就多买点补上差额；币价猛涨，就少买点，甚至卖掉一些“超额”部分。
- 优点：可以看作是一种被动的高抛低吸策略，遇到长期震荡市持币价格会迅速降低。计算也比较简单。
- 缺点：遇到市场暴跌时需要补充大量资金，可能缺乏预算难以执行；遇到市场暴涨时需要卖出大量持币，不利于“囤币党”。我们可以对此作些改进，限制每次投入/卖出的最大金额。

### 何时停止
- 法币本位：比如总投入达到10万美元就停止。
- BTC本位：比如持币数量达到1BTC就停止。
- 策略性止盈：用价值平均策略时，如果市场太给力，持币成本都快成负数了，那就可以考虑止盈离场，开启下一轮定投。

定投最大的敌人不是市场，而是那个半途而废的自己。 关键是选一个你觉得舒服的策略，然后像吃饭睡觉一样坚持下去。

## 使用指南

### 1. 设置参数
- **基础周预算**: 设置每周投资金额
- **时间范围**: 选择模拟的开始和结束日期

### 2. 运行模拟

### 3. 分析结果
- 查看关键指标对比（ROI、BTC累计量、最大回撤等）
- 分析投资组合价值变化趋势
- 比较不同策略的资本效率

## 策略说明

### 标准定投 (Standard DCA)
- 每周固定投资相同金额
- 简单易执行，心理压力小
- 适合长期投资者

### 动态定投 (Dynamic DCA)
- 基于RSI指标调整投资金额
- RSI < 自定义阈值: 增加投资（市场超卖）
- RSI > 自定义阈值: 减少投资（市场超买）
- 平衡风险与收益

### 价值平均法 (Value Averaging)
- 目标：投资组合价值每周增长固定金额
- 低于目标时买入，高于目标时卖出
- 设置买卖限制避免极端操作

## 数据说明

- 包含RSI技术指标计算
